Tokens consumidos
  • --aki-error — fila/pill critico (border-left + text + dot)
  • --aki-warning — fila/pill bajo stock. Texto sobre: --aki-on-warning #000 AAA
  • --aki-info — fila preventa. --aki-on-info #000 5.71:1 PASS AA
  • --aki-success + --aki-on-success #000 — pill OK (sin fila a mostrar en produccion)
  • --aki-border-aa — inputs, botones ghost, icon buttons
  • --touch-target-min 44px — todos los botones de accion

Responsive breakpoints
  • <640px: Card list visible, table display:none. Overflow-x irrelevante.
  • ≥640px: Table visible (min-width 720px dentro de overflow-x:auto). Card list display:none.
  • El wrapper .akb-stock-table-wrap actua como contenedor de scroll horizontal en mobile intermedio (375–640px).
A11y rules
  • Color NO unico diferenciador — border-left + pill texto + label siempre en conjunto.
  • Filtros como aria-pressed toggle buttons (no checkbox ocultos).
  • role="region" + aria-label en el panel contenedor.
  • Botones de sort: aria-label describe la columna y la accion.
  • Empty state con role="status" para lectores de pantalla.
  • Pagination: aria-current="page" en pagina activa, nav aria-label.
