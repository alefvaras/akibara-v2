Decisiones de diseno
  • Float (Variante A): mantener el comportamiento actual — sin cambio de comportamiento per politica.
  • Expanded label (A2): CSS class .akb-wa-float--expanded activable via JS post 3s delay. Opcional.
  • Inline (Variante B): Cell D decide entre rich card (B1) o boton simple (B2). Rich card es mas informativo, simple menos disruptivo.
  • Sticky (Variante C): solo si Cell D quiere reemplazar el float en mobile. NO implementar ambos simultaneamente.

A11y rules
  • Todos los enlaces tienen aria-label descriptivo con nombre del producto cuando esta disponible.
  • rel="noopener noreferrer" en todos los target="_blank".
  • SVG con aria-hidden="true" — el texto o aria-label del anchor provee el nombre accesible.
  • Touch target 44px minimo en todos los botones/enlaces.
  • Focus ring 3px rojo en todos los elementos interactivos.
  • Sticky bar: nav con aria-label.
