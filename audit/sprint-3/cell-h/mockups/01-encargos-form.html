Comportamiento JS / data binding (Cell A implementa) 
// Cell A binds these in PHP shortcode renderer:
//   wp-content/plugins/akibara-preventas/includes/encargos-form.php
//
// 1. JS validation (vanilla, no framework):
//    - email regex client-side + native type="email"
//    - if invalid: add .akb-field--error + render .akb-encargos-form__error
//    - if all valid: submit AJAX to /wp-json/akibara/v1/encargos
//
// 2. AJAX response handling:
//    - success → swap form html with .akb-encargos-form__feedback--success
//    - error   → render .akb-encargos-form__feedback--error + keep form
//
// 3. Loading state:
//    - on submit: button.disabled=true + className+='.akb-btn--loading'
//    - on response: revert
//
// 4. Accessibility:
//    - aria-live="polite" on feedback container (announce to SR)
//    - aria-invalid="true" on .akb-field--error
//    - focus moves to first error field on validation fail
